React 中表单 , 和html中用法有点一样

    input  text
    input  radio
    input  checkbox
    select
    
    
    
    在React中使用表单注意：
    1. value/checked  属性设置后，用户是无法操作的。
    2. textarea 文本域  值建议设置到Value
    3. select的value属性， 值代表选中值，不建议使用option的selected属性
    4. input/textarea/radio/checkbox  会有onChange， 用户每次输入都会触发 （和oninput 事件一样） ， 不是失去焦点的时候触发


    表中在React分类：
    1. 受控表单
        提交数据   onChange, 改变 state
    2.非受控表单
        提交数据 原生  DOM方式，  ref
        给默认值：
            text textarea select  需要写：  defaultValue
            checkbox, radio 需要些： defaultChecked
